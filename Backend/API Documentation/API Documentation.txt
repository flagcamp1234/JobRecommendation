1) Like and unlike Jobs
Method Name: ./like
Method: Get
// request parameters
    url = ./like
    params = 'user_id=' + user_id
    req = {}

Method: Post
//Request Parameters:
    url = ./like
    req = 
    {
      user_id: user_id,
      like: [{
         api_job_id : api_job_id, 
         job_title : job_title,
         location : location,
		 company  : company_name,
         job_snippet : job_snippet
      }]
     }
     
Method: Delete 
// request parameters
    url = ./like
    req = {
      user_id: user_id,
      like: [api_job_id]
    }
